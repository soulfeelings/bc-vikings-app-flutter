# BC Vikings Flutter App

Приложение управления баскетбольной командой BC Vikings, переписанное с React на Flutter.

## Описание

BC Vikings - это система управления баскетбольной командой с функционалом для тренеров и игроков:

### Возможности для игроков:
- 🔐 Вход по логину и паролю
- 📊 Просмотр личной статистики (очки, посещения, уровень)
- 📈 Отслеживание прогресса и уровня
- 📅 История посещений тренировок

### Возможности для тренеров:
- 🔑 Вход по паролю тренера
- 👥 Управление игроками (добавление, редактирование, удаление)
- 📅 Создание и управление тренировками
- ✅ Отметка посещаемости игроков
- 📈 Просмотр статистики команды

## Технический стек

- **Frontend**: Flutter (Dart)
- **State Management**: Provider
- **Backend**: Supabase (PostgreSQL)
- **HTTP Client**: Supabase Flutter SDK
- **Local Storage**: SharedPreferences
- **UI Components**: Material Design 3

## Установка и запуск

### Предварительные требования

1. Flutter SDK (версия 3.27.2 или выше)
2. Dart SDK (включен в Flutter)
3. Веб-браузер для запуска (Chrome рекомендуется)

### Установка зависимостей

```bash
flutter pub get
```

### Запуск приложения

#### Веб-версия (рекомендуется)
```bash
flutter run -d web-server --web-hostname 0.0.0.0 --web-port 8080
```

#### Запуск в браузере Chrome
```bash
flutter run -d chrome
```

### Сборка для продакшена

#### Веб-версия
```bash
flutter build web
```

Собранные файлы будут находиться в папке `build/web/`.

## База данных

Приложение использует Supabase как backend. Структура базы данных включает:

- Автоматическое обновление timestamp'ов
